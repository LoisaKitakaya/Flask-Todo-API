Todo API


Simple Todo backend for your desired frontend


Create, read, update, and delete todos all from an easy-to-use API. Includes user account functionality. Just create a user account and you should be able to generate a JWT (Jason Web Token) access token. Use this token to send requests to the api endpoints.


The purpose of this API is one, to give developers an opportunity to play around with token based authentication. Second, is to offer developers the freedom of designing their own frontend applications without worrying much about what's under the hood.


There is currently a limit of 100 general todos per user account.


To know more about this project check out the source code on GitHub. Sign up today!
