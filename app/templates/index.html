Todo API

Simple Todo backend for your desired frontend


Create, read, update, and delete todos all from an easy-to-use API. Includes user account functionality. Just provide an email and you should receive your api key in your inbox a few moments later. To know more about this project check out the source code on GitHub. Sign up today!


The purpose of this API is one, to give developers an opportunity to play around with bearer token authentication. Second, is to offer developers the freedom of designing their own frontend applications without worrying much about what's under the hood. There is currently a limit of 100 general todos per user account.
